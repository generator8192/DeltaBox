Delta Box is a Command Line Interface (CLI) tool designed for Linux systems, enabling the fully autonomous procedural synthesis of three-dimensional geometry and automated compilation of BSP map files for the Half-Life 1.

Features:
 Mutation Matrix: The `Strangeness` variable accurately acts as a cyclic shift offset, altering seed characters to physically mutate map layers.
 Perlin Noise: Generates natural, endless undulating topography and bedrock rifts instead of primitive flat arenas.
  
Execution Pipeline:
 Compile your procedural world instantly from the terminal:
 bash
  make map
